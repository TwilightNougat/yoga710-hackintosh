## 安装
### BIOS设置：
- 关闭安全启动
- 开启intel XGS
### 引导
- 分区必须大于300Mb
- EFI已配置多系统，不会导致win系统主板编号错误
### 硬件
- 网卡需更换dw1820a，并且屏蔽针脚
- 或者更换intel网卡，注意驱动需要更换
